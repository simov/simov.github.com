            <h2><a href="https://github.com/simov/express-admin">Express Admin</a></h2>
            <p>MySQL admin for NodeJS.</p>
        </header>
 
        <p>I was using Django to administrate a few MySQL databases and since I found NodeJS I thought it would be cool if I have my admins written in Javascript.</p>
 
        <p>I didn't found any solution at that time and eventually I ended up writing my own simple script.</p>
 
        <p>Of course I can't write a framework, an ORM and etc. all by myself and expect it to be competitive and maintainable. Therefore my requirements were to write a <em>simple</em> and <em>maintainable</em> solution.</p>
 
        <p>You can checkout the <a href="http://simov.github.io/express-admin-site/">documentation</a> page. I'm constantly trying to improve it as well as the <a href="https://github.com/simov/express-admin-examples">examples</a> so the most up to date information should be found there.</p>

        <header>
            <h2><a href="https://github.com/simov/styler">Styler</a></h2>
            <p>Chrome styling extension.</p>
        </header>
 
        <p>I was looking for an easy way to reload my pages on file changed while I'm developing styles for injection into web sites (like the monokai theme for <a href="https://gist.github.com/simov/3716262">GitHub's code view</a>). And while there are plenty of chrome extensions for injecting styles into pages I didn't found one that allow me to auto refresh my pages while developing them in Sublime Text.</p>
 
        <p>Anyway I was somewhat familiar with chrome's extension development basics and I was pretty sure that I can hook up the os specific stuff pretty fast using nodejs and web sockets.</p>
 
        <p>Lastly the project doesn't have any tests at the moment but alot of stuff is documented into a trello board so hopefully when the tests are ready they will cover most of the bugs.</p>
    </article>
